# 📚 SemantraNet – Semantic Network Analysis for Literary Works

SemantraNet adalah aplikasi analisis jejaring semantik berbasis multi karya sastra yang memanfaatkan pendekatan Social Network Analysis (SNA) untuk mengungkap keterkaitan makna antar konsep dalam teks sastra.

Aplikasi ini dikembangkan untuk mendukung kajian intertekstualitas, eksplorasi konsep dominan, serta analisis hubungan antar karya secara sistematis berbasis data.

---

## 🚀 Fitur Utama

- 🔍 Ekstraksi konsep dari teks sastra
- 🌐 Pembentukan jejaring semantik (semantic network)
- 📊 Analisis struktur jejaring:
  - konsep dominan
  - konsep penghubung (bridging concepts)
  - pola relasi antar konsep
- 🔗 Analisis keterkaitan antar karya:
  - shared concepts
  - intertekstualitas
- 📁 Ekspor hasil ke format graf (Gephi-ready)

---

## 🧠 Use Case

Aplikasi ini dapat digunakan untuk:

- Analisis karya sastra tunggal maupun multi karya
- Studi intertekstualitas dalam sastra
- Penelitian kolaboratif antara ilmu sastra dan informatika
- Eksplorasi konsep tematik dalam korpus teks
